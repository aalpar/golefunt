# Bibliography

## Primary References

### Software Manual for the Elementary Functions

Cody, W.J. and Waite, W. (1980). *Software Manual for the Elementary Functions*. Prentice-Hall, Englewood Cliffs, NJ. ISBN: 0-13-822064-6.

The foundational text describing algorithms for computing elementary functions and the methodology for testing their accuracy using mathematical identities.

### ELEFUNT Test Package

Cody, W.J. (1988). Algorithm 665: MACHAR, a subroutine to dynamically determine machine parameters. *ACM Transactions on Mathematical Software*, 14(4), 303-311. https://doi.org/10.1145/50063.51907

Describes the MACHAR subroutine for dynamically determining floating-point arithmetic system parameters, essential for portable accuracy testing.

### Complex Elementary Functions

Cody, W.J. (1993). Algorithm 714: CELEFUNT: A portable test package for complex elementary functions. *ACM Transactions on Mathematical Software*, 19(1), 1-21. https://doi.org/10.1145/151271.151272

Extends the ELEFUNT methodology to complex-valued elementary functions. Describes how ratio-based identity testing applies to complex arithmetic, where functions must satisfy identities in both real and imaginary components.

Technical Report: Cody, W.J. (1990). *CELEFUNT: A portable test package for complex elementary functions*. ANL-91/1, Argonne National Laboratory. https://doi.org/10.2172/6310253
